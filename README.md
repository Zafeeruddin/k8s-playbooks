## k8s-playbooks
Collection of Ansible playbooks to deploy and manage Kubernetes-native applications. 
Useful for DevOps engineers or SREs looking to automate k8s setups.

### Usage

Clone the repo:

```bash
git clone https://github.com/username/k8s-playbooks.git
cd k8s-playbooks
```
